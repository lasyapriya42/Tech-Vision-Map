Git and GitHub Roadmap

Introduction to Git

Git is a distributed version control system used to track changes in source code during software development. Learn the basics at Git Documentation.

Introduction to GitHub

GitHub is a web-based platform for hosting and sharing Git repositories. Explore its features at GitHub Docs.

Basic Git Commands

  • git init: Initialize a new Git repository.
  • git clone: Clone an existing repository.
  • git add: Stage changes for the next commit.
  • git commit: Record staged changes.
  • git push: Push changes to a remote repository.
  • git pull: Fetch and merge changes from a remote repository.
  • git branch: List, create, or delete branches.
  • git merge: Merge branches together.
  • git log: View the commit history.

GitHub Workflow

Understand GitHub workflows, including pull requests and issue tracking. Read more at GitHub Flow.

Advanced Git Features

  • Rebasing: Modify commit history with git rebase. Learn more at Git Rebase Documentation.
  • Stashing: Temporarily store changes with git stash. Learn more at Git Stash Documentation.
  • Cherry-Picking: Apply specific commits with git cherry-pick. Learn more at Git Cherry-Pick Documentation.

GitHub Collaboration

Collaborate with teams using features like pull requests, forks, and project boards. Learn more at GitHub Collaboration Docs.
