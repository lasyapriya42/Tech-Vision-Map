HTML

1. What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a webpage using a system of elements and tags.

2. Basic Structure of an HTML Document

An HTML document consists of the following structure:

<html>, <head>, <body> are the basic building blocks.

3. HTML Elements and Tags

Common tags include <h1> to <h6>, <p>, <div>, and <span>. Text formatting uses tags like <b>, <i>, <u>, <em>, <strong>. Semantic tags include <header>, <footer>, <main>, <article>, and <section>.

4. HTML Attributes

Attributes provide additional information about elements. Global attributes include id, class, and style. Specific attributes like src for images and href for links add functionality.

5. HTML Forms

Forms use elements like <form>, <input>, <textarea>, <button>, <select>, and <option>. Key attributes are action, method, placeholder, and required.

6. Multimedia in HTML

Use <img> for images, <audio> for sound, and <video> for videos. Each has attributes like src, alt, controls, autoplay, and loop.

7. HTML Tables

Tables are structured with <table>, <thead>, <tbody>, <tr>, <td>, and <th>. Attributes like border, cellspacing, and cellpadding control appearance.

8. HTML Links and Navigation

Use <a> with href for links. Create navigation with <nav>. Anchor links and external links are common.

9. HTML Lists

Ordered lists use <ol> and <li>. Unordered lists use <ul> and <li>. Description lists use <dl>, <dt>, and <dd>.

10. HTML5 Features

HTML5 introduces new elements like <article>, <section>, and <aside>, as well as features like <canvas> for graphics and <iframe> for embedding content.

11. Best Practices

Write semantic and accessible HTML. Use proper indentation, nesting, and comments (e.g., <!-- Comment -->).
