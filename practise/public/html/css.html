CSS Technology

1. What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout and design of web pages.

2. CSS Selectors

- Basic selectors: *, element, class, id.
- Grouping selectors: ,.
- Attribute selectors: [attribute=value].
- Pseudo-classes: :hover, :nth-child().
- Pseudo-elements: ::before, ::after.

3. CSS Box Model

The box model consists of the following elements:
- Margin
- Border
- Padding
- Content

Inspect and debug the box model using developer tools in modern browsers.

4. CSS Units and Values

- Absolute units: px, cm, mm.
- Relative units: %, em, rem, vh, vw.

5. CSS Styling Basics

- Text styling: font-size, font-family, color, text-align, line-height.
- Backgrounds: background-color, background-image, background-size.
- Borders: border-width, border-radius, border-style.

6. CSS Layout Techniques

- Display properties: block, inline, inline-block, none.
- Positioning: static, relative, absolute, fixed, sticky.
- Flexbox: display: flex; and related properties like justify-content, align-items.
- CSS Grid: display: grid; and grid-template-columns, grid-template-rows.

7. Responsive Design

- Media queries: @media.
- Viewport meta tag.
- CSS units for responsiveness: vh, vw, %.

8. CSS Animation and Transitions

- Transitions: transition-property, transition-duration.
- Keyframe animations: @keyframes and properties like animation-name, animation-duration.
